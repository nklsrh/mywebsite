SUPER CRICKET

Made in 8 weeks for my Industrial Technology - Multimedia HSC subject, it's a fully interactive walkthrough of the fictional 'AXON Racing Team' and its Tech Centre, Garage as well as a private Testing Circuit.

You will be able to pilot the AXR.1 experimental hovercar prototype using a real-world Logitech Driving Force GT racing wheel which includes Force-Feedback effects for a great driving experience.

It includes a 'Time Trial' styled game mode, where you have to beat a set target time. There is also a short trophy presentation if you beat the challenge.

All the assets were created by me in 3dsMax Design '10, Inkscape, Photoshop CS5. The project was coded within the XNA framework using C#.
